# LLMs.txt

We host an [llms.txt](https://razorpay.com/docs/llms.txt) file which instructs AI tools and agents how to retrieve the plain text versions of our pages.

The llms.txt file is an emerging standard for making websites and content more accessible to LLMs. It provides a structured index of our documentation that AI assistants can use to better understand and integrate with Razorpay APIs.

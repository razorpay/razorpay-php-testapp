# Settlements APIs

You can use the Settlement APIs to perform various actions. You can perform a few of these actions from the Dashboard as well.

## List of Settlements API

The table below lists the various Settlement APIs and gives a brief description of each API:

API | Description
---
[View All Settlements ](@/Applications/MAMP/htdocs/new-docs/llm-content/api/settlements/fetch-all.md) | API to view all settlements.
---
[View Settlements by Id](@/Applications/MAMP/htdocs/new-docs/llm-content/api/settlements/fetch-with-id.md) | API to view settlement details by providing the settlement id.
---
[Settlement Recon](@/Applications/MAMP/htdocs/new-docs/llm-content/api/settlements/fetch-recon.md) | API to view all transactions such as payments, refunds, transfers and adjustment settled to you for a particular day or month.

List of Instant Settlements APIs: 

Term | Description
---
[Create Instant Settlements ](@/Applications/MAMP/htdocs/new-docs/llm-content/api/settlements/instant/create.md) | API to create an Instant Settlement.
---
[Fetch all Instant Settlements ](@/Applications/MAMP/htdocs/new-docs/llm-content/api/settlements/instant/fetch-all.md) | API to view all Instant Settlements.
---
[Fetch Instant Settlements by id](@/Applications/MAMP/htdocs/new-docs/llm-content/api/settlements/instant/fetch-with-id.md) | API to view Instant Settlement details by providing the settlement id.
---
[Fetch all Instant Settlements with Payout Details](@/Applications/MAMP/htdocs/new-docs/llm-content/api/settlements/instant/fetch-all-payout.md) | API to retrieve payout details as part of the response for all instant settlements requests.
---
[Fetch Instant Settlements with Payout ID](@/Applications/MAMP/htdocs/new-docs/llm-content/api/settlements/instant/fetch-with-id-payout.md) | API to retrieve payout details as part of the response for a specific instant settlement request.

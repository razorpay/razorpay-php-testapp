# Payments APIs

You can use the Payments APIs to perform various actions. You can perform a few of these actions from the Dashboard as well. 

## List of Payments APIs

The table below lists the various Payments APIs and gives a brief description of each API:

 API | Description
 ---
 [Capture a payment](@/Applications/MAMP/htdocs/new-docs/llm-content/api/payments/capture.md) | API to capture a payment
 ---
 [Fetch a payment](@/Applications/MAMP/htdocs/new-docs/llm-content/api/payments/fetch-with-id.md) | API to view details of a payment
 ---
 [Fetch multiple payments](@/Applications/MAMP/htdocs/new-docs/llm-content/api/payments/fetch-all-payments.md) | API to get details of all payments
 ---
 [Fetch payments based on orders](@/Applications/MAMP/htdocs/new-docs/llm-content/api/payments/fetch-payments-orders.md)|  API to get payments based on orders
 ---
 [Fetch card details for a payment](@/Applications/MAMP/htdocs/new-docs/llm-content/api/payments/fetch-card-details-payment.md)| API to get the card details for a payment
 ---
 [Update a payment](@/Applications/MAMP/htdocs/new-docs/llm-content/api/payments/update.md) | API to update notes of a payment
 ---
 [Payment Downtime API](@/Applications/MAMP/htdocs/new-docs/llm-content/api/payments/downtime.md) | API for payment downtime and webhooks

# View Reports

You can generate the Standard Payment Links report from the Dashboard. 

To generate reports:

1. Log in to the Dashboard.
1. Navigate to **Reports**.
1. On the Reports page, enter the following details:
    1. **Select Report Type**: Select the **Payment Links** report.
    1. **Select Period**: Select the period for which the report should be generated.
    1. **Select Format**: Select the file format. You can report in CSV, XLS and XLSX formats.
    1. **Email Report To**: You can send the report to the email address registered with Razorpay.
1. Click **Generate Report**.

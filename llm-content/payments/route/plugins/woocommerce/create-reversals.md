# Create Reversals

Reverse transfers from the WordPress Admin Dashboard. You can also view payments, transfers and reversal details.

## Create Reversals From WordPress Admin Dashboard

To create a reversal:

1. Log in to the **WordPress Admin Dashboard** and navigate to **Razorpay Route woocommerce** → **Transfers**.
2. Click the required **Transfer Id** to create a reversal.
3. Click **Create Reversal**.
    ![](/docs/assets/images/woocommerce-route-reversals.jpg)
4. Enter the **Reversal Amount** and click **Create Reversal**.
    ![](/docs/assets/images/woocommerce-route-reversals-create.jpg)

The amount is reversed. You can see the details in the **Reversals** tab.

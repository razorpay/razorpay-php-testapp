# How Payment Button Works

Given below is a complete end-to-end flow about how you can use Razorpay Payment Button to accept payments.

![](/docs/assets/images/payment-button-payment-button-working.jpg)

## Payment Button Workflow

To use Payment Button to accept payments:

### 1. Sign Up for a Razorpay Account

To create a Payment Button, you first need to set up your [Razorpay account](@/Applications/MAMP/htdocs/new-docs/llm-content/payments/set-up.md). If you already have an account, log in to the Dashboard and click **Payment Button**.

### 2. Select a Template or Create from Scratch

Click **Create Payment Button** and select a template. Know more about [templates](@/Applications/MAMP/htdocs/new-docs/llm-content/payments/payment-button/quick-pay/#step-1-select-a-template.md).

### 3. Add Button Details

Add the button details. For example, you can add title, button type and so on. Know more about [adding button details](@/Applications/MAMP/htdocs/new-docs/llm-content/payments/payment-button/quick-pay/#21-button-details.md).

### 4. Add Customer Details 

Configure the input fields. For example, you can add email details, add a name field and so on. Know more about [adding customer details](@/Applications/MAMP/htdocs/new-docs/llm-content/payments/payment-button/quick-pay/#22-customer-details.md).

### 5. Embed Payment Button on your Website and Start Accepting Payments

Add the hyperlink to your website. When customers click on the **Pay** button, they will be redirected to the Razorpay Payment Gateway to complete the payment. Know more about [embedding the Payment button to a website](@/Applications/MAMP/htdocs/new-docs/llm-content/payments/payment-button/quick-pay/#embed-payment-button-code-in-website.md).

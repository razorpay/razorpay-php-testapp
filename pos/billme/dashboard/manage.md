# Overview

The **Manage** section in Razorpay Billme brings all key configuration and administrative tools under one unified space. From assigning user roles to handling customer complaints, this hub helps you keep your operations organised and secure.

With **Manage**, you can:

- Control user access levels
- Segment customers based on behavior and data
- Manage multiple store setups
- Organise creatives in a central media bank
- Address and resolve customer complaints
- Update system-wide settings from one place
